# PRD: RYE-K K-Culture Center

국악 교육기관 회원·강사·출석·수납·기관 통합 관리 PWA. 엑셀·메모장 → 실시간 DB 통합.

## 역할
| 역할 | 설명 |
|------|------|
| admin | 원장. 전체 시스템 관리 |
| manager | 행정. 회원·수납·공지 처리 |
| teacher | 강사. 담당 회원 출석·레슨노트 |
| member | 회원/학부모. 출석·레슨노트·공지 열람 |

## 핵심 기능

**1. 회원 관리** — 등록·수정·삭제(소프트), 사진(Canvas압축→Base64), 과목별 강사·요일·시간(`lessons[]`), 상태(재원/휴원/퇴원), 회원코드 RK+4자리, 수강료 강사 차단

**2. 강사 관리** — 등록·수정, 초기비번 연락처 뒷4자리, 30일 재인증, 강사엔 연락처 마스킹

**3. 출석 관리** — 요일/강사 필터, 출석/결석/지각/보강, 그룹레슨 자동감지(같은 강사+요일+시간), 레슨노트(progress/content/assignment), 기관 참석실인원 입력

**4. 수납 관리** — 월별 납부/미납/부분납, 방법(이체/현금/카드), 일회성 추가청구(강사→관리자 승인), 악기대여비 프리셋, 일괄 수강료 설정(BulkFeeModal), CSV 내보내기

**5. 스케줄** — 주간/월간 캘린더, 임시 일정변경(`rye-schedule-overrides`), 공지 이벤트 연동

**6. B2B 기관 파견** — 학교·주민센터 등 출강 계약관리, 수업/반 편집기(InstClassEditor), 가상회원 아키텍처(기존 컴포넌트 재활용), 계약 D-day 알림(D-30 gold / D-7 red)

**7. 회원 포털** — `/myryk`, 회원코드+생년월일4자리, 다자녀 전환, 출석/레슨노트/공지/수납 열람, paused/withdrawn 차단

**8. 공지** — 고정(pinned), 역할별 타겟, 캘린더 연동, 회원별 개인공지

**9. 분석** — 출석률/재원현황/수납률/악기별·강사별·연령대별 통계, 월별 필터, A4 PDF 출력

**10. 시스템 소식** — 릴리즈 타임라인(SystemNewsView), 주요 업데이트 로그인 후 1회 팝업(UpdatePopup)
