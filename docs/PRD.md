# PRD: RYE-K K-Culture Center

## 목표
국악 교육기관 RYE-K K-Culture Center의 회원·강사·출석·수납·기관 파견을 하나의 PWA에서 통합 관리한다. 엑셀과 메모장에 분산된 운영 데이터를 실시간 동기화 DB로 통합하고, 강사·학부모가 각자의 권한 범위에서 셀프서비스 할 수 있게 한다.

## 사용자

| 역할 | 설명 | 주요 니즈 |
|------|------|---------|
| **admin** (원장) | 전체 시스템 관리자 1명 | 전체 현황 파악, 수납 관리, 강사 관리 |
| **manager** (매니저) | 행정 담당자 | 회원 등록·수정, 수납 처리, 공지 발송 |
| **teacher** (강사) | 악기별 강사 14명 | 담당 회원 출석 체크, 레슨노트 작성 |
| **member** (회원/학부모) | 수강생 및 보호자 | 출석 확인, 레슨노트 열람, 공지 확인 |

## 핵심 기능

### 1. 회원 관리
- 회원 등록·수정·삭제(소프트), 사진 업로드(Canvas 압축 → Base64)
- 과목별 담당 강사·요일·시간 다중 지정 (`lessons[]` 배열)
- 상태 관리: 재원 / 휴원 / 퇴원
- 회원 코드 자동 생성 (`RK` + 4자리, 중복 방지)
- 미성년자 자동 감지 (생년월일 기반, 보호자 연락처 UI)
- 수강료 마스킹: 강사 계정엔 수강료 전면 차단

### 2. 강사 관리
- 강사 등록·수정, 악기별 담당 설정
- 초기 비밀번호: 연락처 뒷 4자리 자동 생성
- 30일 재인증 체계 (admin/manager)
- 강사 계정 → 담당 회원 연락처 전면 마스킹 (개인정보 보호)

### 3. 출석 관리
- 요일별·강사별 필터 출석 체크 (출석/결석/지각/보강)
- 그룹 레슨 자동 감지: 같은 강사 + 같은 요일 + 동일 시간 → 일괄 처리
- 레슨노트 작성 (progress / content / assignment 필드)
- 결석 시 보강 일정 등록 연동
- 기관 파견 수업: 참석 실인원 입력 필드

### 4. 수납 관리
- 월별 수납 현황 (납부/미납/부분납)
- 수납 방법: 계좌이체 / 현금 / 카드
- 일회성 추가 청구 등록 (강사 요청 → 관리자 승인 워크플로)
- 악기 대여비 유연 설정 (DB 기반 feePresets)
- 일괄 수강료 설정 (BulkFeeModal, 과목·강사 필터)
- CSV 내보내기

### 5. 스케줄 관리
- 주간 뷰 / 월간 뷰 캘린더
- 임시 스케줄 변경 (`rye-schedule-overrides`)
- 공지사항 이벤트 날짜 → 캘린더 자동 연동

### 6. B2B 기관 파견 관리 (v12.1+)
- 학교·주민센터·기업 등 외부 출강 계약 관리
- 기관 내 수업/반 단위 편집기 (`InstClassEditor`)
- **Option C 가상회원 아키텍처**: 기관 수업 → 런타임에 가상 student 객체 변환 → 기존 출석/수납/스케줄 컴포넌트에 그대로 주입 (기존 코드 0 수정)
- 계약 D-day 알림: D-30 이내 gold, D-7 이내 red
- 기관 담당자 연락처는 강사에게도 노출 (출강 현장 대응 목적)

### 7. My RYE-K 포털 (회원 전용)
- URL: `?myryk` (또는 `/myryk`)
- 로그인: 회원코드(RK\_\_\_\_) + 생년월일 4자리
- 다자녀 지원: 동일 보호자 연락처로 여러 자녀 전환
- 출석 기록 열람, 레슨노트 열람 + 댓글 작성, 공지사항 확인
- 수납 기록 열람
- paused/withdrawn 회원 접근 차단

### 8. 공지 관리
- 고정 공지(pinned), 역할별 타겟 발송
- 스케줄 이벤트 연동 (날짜·기간 설정 시 캘린더 자동 표시)
- 회원별 개인 공지 (`rye-student-notices`)

### 9. 분석 리포트
- 출석률 / 재원·휴원·퇴원 현황 / 수납률 / 악기별·강사별·연령대별 통계
- 월별 필터 (최근 12개월)
- A4 PDF 출력 (print CSS)

### 10. 시스템 소식 (v14.1+)
- 릴리즈 노트 타임라인 (`SystemNewsView`)
- 주요 업데이트 로그인 후 1회 팝업 (`UpdatePopup`, 3중 조건: 버전·isMajor·role 타겟 일치)

## MVP 제외 사항
- 세금계산서 자동 발행 (수동 체크리스트로 대체)
- 기관 월별 정산 리포트 PDF (v14.x 예정)
- 출강 교통비 항목 (v14.x 예정)
- 알림톡(카카오) 실제 발송 (현재 콘솔 목업)
- React Router (URL param 방식으로 충분)
- 상태관리 라이브러리 (Firestore onSnapshot이 단일 소스)
