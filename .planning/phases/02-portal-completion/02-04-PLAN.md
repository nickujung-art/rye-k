<div style={{marginBottom:16}}>
  <button
    type="button"
    className="btn btn-secondary btn-full"
    disabled={practiceGuideLoading}
    onClick={async () => {
      setPracticeGuideLoading(true);
      setPracticeGuideErr("");
      try {
        const token = await getPortalIdToken();
        const lastNote = sNotes?.[0];
        const res = await fetch("/api/ai/practice-guide", {

    </div>
  )}
</div>

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-04-01 | Spoofing | practice-guide Worker 인증 | accept | Anonymous Firebase token은 유효한 JWT. Worker의 _middleware.js가 서명 검증. 포털 사용자만 호출 가능. |
